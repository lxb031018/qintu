## 📱 发送者端

---

### 底部导航栏结构

发送者端应用采用**底部三 Tab 导航**结构，从左到右依次为：

| Tab | 图标 | 职责 | 页面编号 |
|-----|------|------|----------|
| **主页** | 🏠 | 路径规划、发送导航 | 3、4、5 |
| **绑定** | 🔗 | 管理绑定关系（添加、解绑） | 2 |
| **设置** | ⚙️ | 应用设置、账号管理、帮助 | 7 |

**设计原则**：
- ✅ **职责专一**：每个 Tab 只负责一个核心功能
- ✅ **解绑在绑定页**：解绑操作在"绑定"Tab 中进行，不在设置页
- ✅ **Home 专注核心**：主页只处理路径规划和发送导航，不混杂其他功能

│  │              [解绑]  │   │  ← 单个解绑按钮
│  └─────────────────────┘   │
│                             │

**交互说明**：
- 输入接收者手机号 → 点击"发送绑定请求" → 系统向接收者发送绑定请求
- 接收者确认绑定 → 绑定成功 → 刷新列表
- **解绑操作**：
  - 每个已绑定的接收者右侧有"解绑"按钮
  - 点击"解绑" → 弹出确认弹窗"确定要解绑 xxx 吗？"
  - 确认解绑 → 从列表移除该接收者
- 底部导航栏：主页、绑定（当前）、设置

## 📐 设计规范

### 字体规范

| 元素 | 发送者端 | 接收者端 |
|------|---------|---------|
| 标题 | 20sp | 24sp |
| 正文 | 16sp | 20sp |
| 按钮 | 18sp | 22sp |
| 提示文字 | 14sp | 18sp |

### 颜色规范

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 主色调 | `#4CAF50` | 绿色，代表安全、出行 |
| 辅助色 | `#FF9800` | 橙色，用于提醒、警告 |
| 成功 | `#4CAF50` | 导航完成、绑定成功 |
| 错误 | `#F44336` | 网络错误、操作失败 |
| 背景 | `#F5F5F5` | 浅灰色背景 |
| 卡片 | `#FFFFFF` | 白色卡片背景 |

### 按钮规范

| 状态 | 样式 |
|------|------|
| 默认 | 圆角 8px，主色调背景 |
| 按下 | 颜色加深 10% |
| 禁用 | 灰色 50% 透明度 |
