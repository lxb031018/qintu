# 开发模式使用说明

## 🛠️ 快速查看所有页面

### 方法一：使用模拟登录（推荐）

在 `lib/features/auth/auth_page.dart` 中已启用模拟登录：

```dart
const bool useMockAuth = true; // 开发模式：启用模拟登录
```

**使用方式**：
1. 启动应用
2. 输入任意 11 位手机号（如：`13800138000`）
3. 点击 **🧪 模拟登录（开发模式）** 按钮
4. 直接进入角色选择页面

### 方法二：开发模式配置（快速跳转）

在 `lib/main.dart` 顶部有开发模式配置：

```dart
/// 是否启用开发模式
/// true: 跳过登录，直接显示角色选择页面
/// false: 正常登录流程
const bool devMode = true;

/// 开发模式默认角色（用于快速查看不同页面）
/// 'sender' - 发送者主页
/// 'receiver' - 接收者主页
/// null - 显示角色选择页面
const String? devModeDefaultRole = null; // null = 显示角色选择页面
```

**使用方式**：

| 配置值 | 效果 | 用途 |
|--------|------|------|
| `devModeDefaultRole = null` | 显示角色选择页面 | 查看角色选择UI |
| `devModeDefaultRole = 'sender'` | 直接进入发送者主页 | 查看发送者三Tab架构 |
| `devModeDefaultRole = 'receiver'` | 直接进入接收者主页 | 查看接收者单页架构 |

**示例**：想快速查看发送者主页
```dart
const String? devModeDefaultRole = 'sender';
```

## 📱 页面架构预览

### 1. 角色选择页面
- 两个大按钮：我是接收者 / 我是发送者
- 点击后进入对应主页

### 2. 发送者主页（三Tab架构）
```
┌─────────────────────────┐
│    发送导航指引 (AppBar) │
├─────────────────────────┤
│                         │
│   路径规划内容           │
│   - 起点输入             │
│   - 终点输入             │
│   - 规划路线按钮         │
│                         │
├─────────────────────────┤
│  🏠主页  🔗绑定  ⚙️设置  │ (底部导航)
└─────────────────────────┘
```

### 3. 接收者主页（简洁单页）
```
┌─────────────────────────┐
│  定位开关  (AppBar)      │
├─────────────────────────┤
│                         │
│     🧭 (图标)           │
│                         │
│  等待接收导航指引...     │
│  暂无导航任务             │
│                         │
├─────────────────────────┤
│              ⚙️ 设置     │ (浮动按钮)
└─────────────────────────┘
```

### 4. 设置页面（两个角色共用）
- 角色切换卡片
- 主题设置卡片（浅色/深色/跟随系统）✅
- 退出登录卡片

## 🎨 主题切换

设置页面已完整集成主题切换功能：
- 浅色模式
- 深色模式
- 跟随系统

**测试方法**：
1. 进入任意角色主页
2. 点击设置按钮（发送者在底部Tab，接收者在右下角FAB）
3. 点击主题设置卡片切换主题

## ⚠️ 注意事项

1. **开发模式仅用于开发测试**
   - 使用模拟用户ID和Token
   - 不会调用真实API
   
2. **发布前必须关闭开发模式**
   ```dart
   const bool devMode = false;
   ```

3. **模拟登录的限制**
   - 不会保存真实用户数据
   - 无法测试绑定关系等需要后端的功能
   - 仅用于UI预览和流程测试

## 🚀 快速开始

```bash
# 1. 确保依赖已安装
flutter pub get

# 2. 运行应用
flutter run

# 3. 根据 devModeDefaultRole 配置查看不同页面
```

修改 `devModeDefaultRole` 后热重载（`r`）或热重启（`R`）即可看到效果！
